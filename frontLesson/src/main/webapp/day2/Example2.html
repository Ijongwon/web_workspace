CSS 기본 프로퍼티 정리

번호 속성 설명 예시
1 colors color 속성은 색상,정확히는 글자의 색상 을 의미합니다.
  • red,blue 등 이미 정의된 색
  • #000,#FFFFFF 등의 16진수 색상 코드
  • rgb(255,255,255) 등의 rgb 색상
  • rgba(200,100,150,0.5)등의 알파(투명도)가 적용된 rgba 색상
color: red;
color: #0A0;
color: rgb(0,0,150);
2 width,height 각각 가로 길이, 세로 길이 를 의미한다.
이 속성들은 block 요소에만 설정된다.
상대값(%)를 줄수도 있다. 		width: 100px;
height: 60px;
3 margin,padding 각각 바깥쪽여백, 안쪽 여백 을 의미한다.
width,height 속성과 마찬가지로 숫자 뒤에 단위를 표시하여 적는다.
border를 경계로 나뉜다.
방향은 위부터 시계방향 순이다. 		margin(padding): 20px
margin(padding): 30px 10px
margin(padding): 30px 10px 20px 50px
4 font 글자의 폰트 를 정의하는 속성
세부적인 글꼴 관련 속성은 아래와 같다.
  • font-style : 이탤릭체 등의 글꼴의 스타일 지정
  • font-weight : 글자 두께
  • font-variant : 글꼴 변형(소문자를 대문자로 바꾸는 등의)
  • font-size : 글자 크기
  • font-height : 줄 간격
  • font-family : 글꼴(굴림,돋움,...)
 font-size : 20px;
font-family : 나눔고딕,NanumGothic,돋움,Dotum;
font-weight : bold
5 text-align 텍스트의 정렬 방향 을 의미
  • left : 왼쪽 정렬
  • right : 오른쪽 정렬
  • center : 중앙 정렬
  • justify : 양쪽 정렬(자동 줄 바꿈시 오른쪽 경계선 부분 정리)
 text-align : right;
text-align: justify;
6 background 태그의 배경 을 지정하는 속성
  • background-color : 배경 색
  • background-image : 배경 이미지
  • background-repeat : 배경 이미지 반복여부
  • background-position : 배경 이미지 위치
 background-color : green;
background: yellow url('/images/attach/google.png') no-repeat center center;
7 border 태그의 테두리를 설정 하는 속성
width-style-color의 순서로 사용한다.
테두리의 특정 방향만 따로 설정할 수도 있다.
  • border-width: 테두리의 두께, px 단위 사용
  • border-style: 테두리의 스타일, 실선,점선,이중선...
  • border-width: 테두리의 색상, color 속성의 포맷 사용
 border: 4px dotted green;
border-bottom: 5px solid blue;
8 display 요소를 어떻게 보여줄지 결정 한다.
  • none: 보이지 않음, 영역차지 X
  • block: 블록 박스(div,p,h,li...) 가로영역을 모두 채우며, 줄바꿈이 된 것 처럼 보임
  • inline: 인라인 박스(span,b,i,a) 줄 바꿈 X, width와 height 지정 X
  • inline-block : block과 inline 의 중간형태, 줄바꿈이 되진 않지만 크기 지정 가능
 display : none;
display : inline-block;
